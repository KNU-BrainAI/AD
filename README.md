## Introduction
This repository contains the official implementation of AlzEEGNet.
AlzEEGNet is a novel framework for the diagnosis of preclinical Alzheimer's Disease using resting-state EEG datasets.

### Prerequisites

- Python 3.8.6
- Pytorch 1.6.0
- mne 0.21.0
- sklearn 0.23.2
